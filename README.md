## 🤖 What I'm Building

###  AI Product Photography Platform
> Upload one product photo → get 10 professional studio-quality images in under 3 minutes

- **Hybrid AI pipeline**: Gemini 3 Pro Image for hero shots + Replicate FLUX.1 Kontext for batch angle generation
- **Dynamic angle selection**: Gemini Vision analyses each product and recommends the best angles based on garment type, silhouette, and user's photo count setting
- **61% cost reduction** achieved through provider-agnostic architecture — from $0.67 to $0.26 per 10-angle job
- **LoRA fine-tuning** pipeline planned on FLUX.1 Kontext Dev via Replicate for category-specific photoshoot style training
- Stack: React Native (Expo) · NestJS 11 · PostgreSQL · Redis · BullMQ · AWS S3 + CloudFront · Razorpay · Stripe
